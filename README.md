# ⚖️ BMI Calculator

![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python)
![Tkinter](https://img.shields.io/badge/GUI-Tkinter-green)
![CSV](https://img.shields.io/badge/Data-CSV-orange)
![License](https://img.shields.io/badge/License-MIT-lightgrey)

A modern **BMI (Body Mass Index) Calculator** built using **Python** and **Tkinter**. This desktop application calculates BMI based on the user's weight and height, classifies the BMI category, and stores previous calculations in a history file for future reference.

---

# 🚀 Features

* ⚖️ Calculate Body Mass Index (BMI)
* 📏 Input weight (kg) and height (feet)
* 📊 Automatic BMI classification

  * Underweight
  * Normal Weight
  * Overweight
  * Obese
* 📝 Save BMI history to a CSV file
* 📅 Display the last 10 BMI records
* ✅ Input validation
* ⚠️ Error handling
* 🎨 Simple and user-friendly GUI

# 📚 BMI Categories

| BMI Range      | Category      |
| -------------- | ------------- |
| Below 18.5     | Underweight   |
| 18.5 – 24.9    | Normal Weight |
| 25.0 – 29.9    | Overweight    |
| 30.0 and above | Obese         |
